# 更新说明

## v1.2.13

### 新功能
- 文件复制/移动时支持冲突处理：当目标位置已存在同名文件时，可以选择覆盖（合并目录并替换同名文件）或跳过（保留目标位置已有项目），覆盖写入采用原子操作确保数据安全。

### 优化
- 文件管理器解压压缩包时，超过 5GB 的文件不再尝试通过面板直接解压，改为提示”文件超大，面板解压可能不稳定”，并根据压缩包格式给出对应 SSH 解压命令。
- WordPress 可疑访问分析中的”填入封禁”按钮提示调整为”IP已填入手动封禁IP框，请手动选择封禁时长和点击封禁按钮”，避免误以为已自动封禁。

### 安全
- 保留解压路径越权校验、非法归档条目校验和压缩包文件数量上限，超大压缩包建议走 SSH 端处理，降低面板请求超时或解压不稳定风险。
