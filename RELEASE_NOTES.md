## v1.2.0-rc5

### 安全加固
- **数据库清空操作增加名校验**：清空站点数据库前校验库名格式（仅允许字母数字下划线，最长 64 字符），防止异常输入注入 SQL。
- **Zip 解压增加多层防护**：限制压缩包内文件数量（100000），防止 zip bomb 攻击。补全了文件写入和目录创建的错误处理。
- **登录表单添加 method=POST**：防止 JS 资源加载失败时表单回退为 GET 提交，导致用户名密码暴露在 URL 中。
- **Ed25519 更新签名校验**（#336ea00）：面板内更新新增 Ed25519 独立签名校验。sha256 校验文件由开发者本地私钥签名后上传，面板内置公钥验签。

### 功能改进
- **移除解压体积限制**：取消单文件 512MB 和总 2GB 限制，避免大数据备份恢复时被误拦。
- **分片上传 + 断点续传**：大文件自动 5MB 分片上传，支持页面刷新后断点续传，分片失败自动重试。
- **面板新版本告警**：检测到 GitHub 有新版本时自动邮件通知管理员，更新完成后发送恢复通知。

### Bug 修复
- **修复 XML-RPC 防火墙误封合法流量**：fail2ban 防御规则对 xmlrpc.php 的匹配增加 HTTP 403 状态码限制。
- **修复 Release 二进制缺少静态资源**：alpine.min.js、chart.min.js、main.css 纳入 Git 管理，确保 CI 和本地编译均包含完整前端资源。
