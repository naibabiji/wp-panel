## v1.1.0-beta8

### 过渡版本：数据库归一化

为 v1.2.0 正式版做准备，beta 用户更新此版本后，数据库版本号自动归一到正式基线。

### Bug 修复

- 修复计划任务中 site_id 为空的普通命令任务不写入 cron 的问题
- 修复文件批量压缩在子目录下路径错误导致找不到文件的问题
- 修复文件批量压缩 archiveName 可路径穿越写到站点外的安全问题
- 修复 MariaDB 密码通过命令行传参导致进程列表泄露的问题
- 修复 MariaDB 用户密码含单引号导致 SQL 语法错误的问题
- 修复建站和修改域名时不检查别名与已有站点冲突的问题
- 修复远程备份接口将 SSH 密码明文返回前端的问题
- 修复更新校验 SHA256 下载失败时静默跳过校验的安全问题
- 修复大日志文件全量读入内存导致性能问题
- 修复远程备份设置保存时脱敏标记覆盖真实密码的问题
- 修复主域名可能撞别人别名导致 Nginx 冲突的问题
- 修复手动 cron 任务执行丢失超时保护的问题

### 命令白名单

- 从白名单移除 bash，改为直接调用 exec.Command，白名单更诚实

### 数据库升级机制

- 清空历史升级条目，v1.2.0 以全新系统发布
- beta 版本自动归一化到正式基线 `1.0.0`
- 未知正式版本直接报错，不静默跳过
