# 更新说明

## v1.2.21

### 新增：SSL 证书包下载

- 网站详情页新增“下载证书包”能力，管理员可下载当前站点的 SSL 证书和私钥，用于手动上传到 CDN 后台。
- 证书包包含 `fullchain.pem`、`privkey.pem` 和 `README.txt`，压缩包内不包含服务器绝对路径。
- 下载接口会校验站点 SSL 状态、证书文件存在性和证书目录边界，避免任意文件读取。
- 下载响应增加 no-store 缓存控制，降低证书包被中间层缓存的风险。

### 新增：配套插件受控读取 SSL 证书

- 网站详情页新增“允许配套插件读取 SSL 证书”开关，默认关闭。
- 新增站点级 SSL 证书导出接口，供本机 WordPress 插件在获得站点 API Key 且管理员显式授权后读取证书。
- 插件导出接口限制为本机回环地址访问，并校验 `X-WP-Panel-Key`、站点域名、SSL 状态和站点授权开关。
- 导出接口返回证书、私钥、域名、别名和到期时间，便于后续独立 CDN 插件对接使用。

### 数据库升级

- `websites` 表新增 `ssl_export_enabled` 字段，用于控制单站点证书导出权限。
- 老版本升级后该字段默认关闭，已安装服务器不会自动开放证书读取能力。
- 新装和升级路径均已补充字段处理，升级步骤可重复执行。

### 安全与审计

- 证书导出操作记录到操作日志，日志只记录结果摘要，不记录证书、私钥或服务器路径。
- 插件接口默认拒绝跨站点 API Key 读取，站点 A 的 API Key 不能读取站点 B 的证书。
- 非法域名、未启用 SSL、证书文件缺失、未开启导出权限等场景均返回明确错误。

### 测试

- 增加 SSL 证书包下载测试，覆盖未启用 SSL、不存在站点、证书文件缺失和 zip 内容结构。
- 增加插件证书导出安全测试，覆盖默认关闭、开关切换、跨站点 API Key 隔离和非法域名。
- 增加数据库升级测试，覆盖从旧版本补充 `ssl_export_enabled` 字段以及重复执行安全。
