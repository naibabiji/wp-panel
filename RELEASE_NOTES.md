## v1.2.0-rc2

### 安全防护
- **Cloudflare 代理站 Nginx 层封禁**：Cloudflare real_ip 还原真实访客 IP + Nginx geo 黑名单 + fail2ban wppanel-nginx action 双通道联动。被禁 IP 在 Cloudflare 代理下也能有效拦截（444 关闭连接），弥补 nftables 在 CDN 场景下失效的问题。
- **站点级 XML-RPC 防护**：默认禁用 xmlrpc.php，Nginx 直接返回 403 不经过 PHP-FPM。WordPress 优化卡片新增开关，配套插件显示状态与影响说明。
- **fail2ban 改用 reload 替代 restart**：修改安全设置不再清零已有封禁计数，解决攻击窗口期 fail2ban 被反复重启导致封禁失效的问题。
- **文件备份排队锁**：多站点同时触发文件备份时按序执行，避免并发 mysqldump/tar 争抢磁盘和 CPU。含 PID 死锁检测自动清理。

### Bug 修复
- **修复跨域名数据库名冲突**：`buildSiteName` 错误丢弃 TLD 后缀，`test.com` 和 `test.cn` 生成相同数据库名。
- **SSL 证书申请失败显示具体原因**：之前仅返回通用提示，现透传具体错误（DNS 未解析、频率限制等）。
- **修复解压 zip 包时误报"非法路径"**：多层目录父目录未创建时 `resolvePathForAccess` 解析失败。
- **`wp password` 命令第二道密码静默失败**：SQLite 更新未检查 `db.Exec` 错误。
- **CLI 子命令不触发全站 Nginx/FPM 重建**：避免重置密码时无意义遍历全站。
- **修复勾选告警规则时误报 SMTP 配置错误**：前端将 SMTP 配置混入告警规则保存。
- **修改 BasicAuth 密码后浏览器不弹窗**：`updateConfigValue` 未同步更新内存缓存。
- **面板设置保存 BasicAuth 静默失败**：`updateConfigValue` 所有错误路径均静默返回。
- **全站 Nginx 重建修复**：证书路径和模板版本改为从数据库读取，不再硬编码。
- **到期时间编辑修复**：日期格式兼容 `<input type="date">`，`db.Exec` 增加错误检查。
- **远程备份修复**：计划任务文件备份上传协程被进程退出杀掉，改为同步上传。remote_path 为空时补齐默认值、配置缺失写日志不再静默跳过。数据库备份同理改为同步上传。
- **数据库自动备份调度器跳过当天 4 点**：面板启动时间在 4 点之前时，调度器错误地排到第二天 4 点，导致当天备份被跳过。改为先判断今天 4 点是否已过再决定。
- **CPU 使用率计算为开机平均值**：/proc/stat 是开机累计值，之前单次 idle/total 得到的是历史平均值，CPU 跑满也显示 20-30%。改为两次差值计算实时值，仪表盘和采集器均已修复。
- **告警未真正「持续 5 分钟」**：CPU 和内存告警之前一次超标即触发，现已实现持续 5 分钟才发送，中途恢复重置计时。
- **告警邮件/日志时间显示 UTC**：monitoring_metrics 存 UTC 时间，告警消息直接使用，改为转系统本地时区显示。
- **SMTP 配置保存覆盖告警规则**：SMTP 组件保存时全量提交所有 settings，覆盖告警规则和 Webhook 配置。限定为 6 个 SMTP 字段，互不污染。
- **smtp_host 接受 true/false**：防御性校验，拒绝布尔值写入 SMTP 地址。

### 新功能
- **仪表盘新增"网站资源"卡片**：按 CPU 排序显示各站点 PHP-FPM 进程实时资源占用，高负载高亮，点击域名直达详情页。
- **网站详情页支持设置到期时间**：基本信息卡片新增到期时间显示与编辑。
- **配套插件新增 XML-RPC 状态显示**：WordPress 后台可查看开关状态和影响范围说明。

### 优化
- **内存告警文案改为「可用内存 < 10%」**：比「内存 > 90%」更直观易懂，告警页面、邮件、日志、数据库种子描述全部统一。
- **建站默认插件新增 Redis Cache**：建站时 redis-cache 插件默认勾选，用户无需手动安装即可启用 Redis 对象缓存。
- **网站详情页布局重组**：卡片重新分组（基本信息 | 监控+数据库 / SSL+Nginx | WordPress优化 / 日志全宽），卡片合并，视觉均衡。
- **网站列表"备份"→"DB备份"**：明确该列对应数据库自动备份。
- **计划任务手动执行即时反馈**：点击执行按钮立即显示"执行中"，无需手动刷新。
