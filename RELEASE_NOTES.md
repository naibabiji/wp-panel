## v1.2.0-rc4

### 安全防护
- **更新签名校验（Ed25519）**：面板内更新新增 Ed25519 独立签名校验。sha256 校验文件由开发者本地私钥签名后上传，面板内置公钥验签。即使 GitHub Release 被篡改，攻击者无法伪造签名，面板拒绝更新。

### Bug 修复
- **修复 XML-RPC 防火墙误封合法流量**：fail2ban 防御规则对 xmlrpc.php 的匹配增加 HTTP 403 状态码限制。旧规则封禁所有 xmlrpc.php 访问，若站点开启 XML-RPC（如 Jetpack 依赖），合法请求也会被封。新规则仅在 XML-RPC 已关闭仍被请求时才触发封禁。
