# CLAUDE.md — WP Panel

## 项目

WP Panel — WordPress 专用服务器管理面板，Go + Gin + SQLite + TailwindCSS + Alpine.js + Chart.js。
目标环境: Debian 13, PHP 8.3, Nginx 1.26, MariaDB, Redis 8.0。
不做向下兼容，按目标环境最新特性开发。

## Karpathy 准则

### 1. 先想清楚再动手
- 不确定时先问，不要猜测。
- 有多种方案时先列出来，不要默认选一个。
- 有更简单的方案就说出来。

### 2. 只写必要的代码
- 不添加需求之外的功能。
- 不为单次使用创建抽象。
- 不处理不可能发生的错误场景。
- 200 行能写完的不要写成 50 行 — 但同样的，50 行能写完的不要写成 200 行。

### 3. 精准修改
- 只改任务相关的代码，不顺手"优化"旁边的代码、注释、格式。
- 不改没坏的东西。
- 匹配现有代码风格。

### 4. 目标驱动
- 编译通过 + go vet 通过 = 完成基准。
- 多步骤任务先列步骤再执行。
